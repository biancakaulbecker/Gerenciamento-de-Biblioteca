# API Loja - Sistema de Estoque

## Tecnologias
- Node.js
- Express
- HTML5/CSS3/JS (Fetch API)

## Como rodar
1. `npm install`
2. `npm start`
3. Acesse `http://localhost:3000`

## Endpoints
- `GET /api/produtos` - Lista todos
- `POST /api/produtos` - Cria um produto
- `DELETE /api/produtos/:id` - Remove
